Responsive design

Qu'est-ce que le responsive design ?

Le responsive design (ou design adaptatif) est une approche de conception web qui permet à un site de s'adapter automatiquement à la taile de l'écran.

Les breakpoints

Un breakpoint est un seuil de largeur d'écran à partir duquel on modifie l'apparence de la page afin de l'adapter au support (mobile, tablette, desktop)

Liste des breakpoints

  • Mobile => max-width: 480px
  • Tablette => max-width: 768px
  • Laptop (ordinateur portable) => max-width: 1024px
  • Desktop (ordinateur fixe) => min-width: 1025px

Attention !!!!

La résolution des écrans en css est représentée en pixel logique

Dans la réalité les appareils sont en pixel physiques

Par exemple, un Iphone 13 a une résolution logique de 390 x 844 pixels mais en réalité il a une résolution physique de 2532 x 1178 pixels

Par exemple, pour un écran Retina 3 pixels physiques pour 1 pixel logique

Principes fondamentaux du responsive design

Utilisation des % ou vh/vw au lieu de valeurs fixes comme le pixel (px)

Utiliser des propriétés telles que max-width: 100% pour que les images ne dépassent jamais leur conteneur

Les media queries => Ce sont des instructions qui appliquent des styles en fonction de la taille de l'écran

Actuellement nous vivont dans un monde où les téléphones portables sont omniprésents, il est conseillé MAINTENANT de concevoir son site en MOBILE FIRST

Le mobile first est un concept de web design optimisé pour le mobile, il consiste à concevoir un site en mettant la priorité sur la version mobile et ensuite d'adapter le site progressivement sur des écrans plus larges

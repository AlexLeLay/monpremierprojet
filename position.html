Positions et overflow

Les positions

Introduction

La propriété CSS position permet de contrôler la manière dont un élément est positionné dans le document.

Elle détermine si un élément suit le flux normal du document ou s'il est placé précisément avec les propriétés top, right, bottom, left

1. Position statique

Comportement: L'élément suit le flux normal, position par défaut de tous les éléments HTML

top, right, bottom, left n'ont aucun effet si l'élément a cette propriété

Exemple:

        
            .static {
                height: 50px;
                width: 50px;
                background-color: blue;
                position: static;
            }

JE SUIS COLLANT !!

2. Position relative

Comportement: L'élément est positionné par rapport à sa position normale

Il occupe toujours sa place dans le flux du document

Les propriétés top, right, bottom et left, le déplacent visuellement, sans changer sa place d'origine

        
            .relative {
                height: 50px;
                width: 50px;
                background-color: blue;
                position: relative;
                bottom: 2rem;
                left: 3rem;
            }

3. Position asbolue

L'élément est retiré du flux normal

Il est positionné par rapport au premier parent positionné(non static)

Si aucun parent n'est positioné, il se basera sur la balise "html"

4. Position Fixe

L'élément fixé est est rétiré du flux comme absolute

Mais il est positionné par rapport à la fenêtre du navigateur

Il ne bouge pas quand on scroll

ELEMENT FIXÉ

5. Position sticky

Le comportement est un mixte entre relative et fixed

L'élément reste dans le flux jusqu'à ce qu'on scrolle au point où il devient fixe
